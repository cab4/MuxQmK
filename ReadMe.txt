Projeto se propõe a minimização de multiplexadores usando minimização de funções Booleanas.

Aqui, faremos:
- Geração de vetores reais a partir do produto tensorial de estados emaranhados, cujo resultado é um vetor a que se deseja obter através de alguma técnica
  de Preparação de Estado Quântico (QSP - Quantum State Preparation);
- A partir do vetor gerados, aplicaremos alguma técnica de QSP para obter árvore de ângulos.
- A partir da árvore de ângulos, usaremo minimização de funções Booleanas, que inicialmente será feita usando mapa de Karnaugh, mas que pode ser trocada por 
outra técnica dependendo da complexidade para maiores dimensões.
- Várias rotinas serão criadas, nem todas atenderão a casos gerais, pois alguns casos são mais urgentes para validações.
